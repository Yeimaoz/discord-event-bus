# Roadmap

## v0.1.0 (current)
Publish-only library. Sync + async.

## v0.2.0 (planned)
- Subscribe API + bot daemon (Discord Gateway WebSocket)
- Manifest `direction = subscribe` honored

## v0.3.0
- Manifest-driven channel routing rules (severity → channel)
- Multi-channel routing logic

## v0.4.0+
- Slash command framework (platform-registered handlers)
- Multi-Discord-server support (server_id in manifest)
